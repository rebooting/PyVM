
# PyVM toy

This is a toy implementation of a simple virtual machine using Python. 

## Objective
The objective of this project is to create a simple virtual machine that can execute a basic imaginary instruction set. 

it's meant to be an evening's practice project with autocomplete turned off, so that I have to code without relying on the IDE's suggestions.
