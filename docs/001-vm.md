
# set up VM
- stack
- memory : 1024 bytes
- instruction pointer
- registers : 2 registers, R0 and R1


# VM behaviour

program loads from 0 upwards
stack grows downwards from 1023 

# Registers
R0 : general purpose register, also used as push-source and pop-destination for stack operations
R1 : general purpose register

# instruction set
0 - NOP ( does nothing )
1 - PUSH ( R0 as a push-source )
2 - POP ( R0 as a pop-destination )
3 - PRINT < register, where 0 is R0 and 1 is R1 >
4 - MOV <register> <value>
99 - HALT
